CSS Cheatsheet

Selectors

Formula Example Type Description
element { } body { } Element Selector This one you can select any html element
.class { } .title { } Class Selector This one you can select any html class
#id { } #name { } Id Selector This one you can select any html id
element1, element2 { } h1, .table { } Multiple Selector This one you can select more than one for style
element(parent) element(children) { } div h1 { } Descendent Selector This one you can select elements that are descendent from another

Visual Rules

Formula Example Type Description
text-align: position; text-align: center; The Text Position This one determines the position of the text
color: color; color: #6b5a5a; The Color This one determines the color of element selected
font-size: unit of measurement; font-size: 30px; The Size of the Text This one determines the size of the font of element selected
background-color: color; background-color: blue; The Background Color This one determines the background color of element selected
